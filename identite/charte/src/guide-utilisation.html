Comment utiliser correctement les fichiers de l'identité visuelle — v1.0, juillet 2026

1. Les fichiers livrés

DossierContenu
identite/logo/svg12 logos vectoriels : 3 dispositions (icône, horizontal, vertical) × 4 versions (couleur, monochrome, noire, blanche)

2. Quelle version choisir ?

SituationFichier à utiliser
Fond blanc ou très clair (site, documents)logo-horizontal-couleur
Fond bleu, photo, fond foncélogo-horizontal-blanc (ou vertical)
Espace étroit ou carré (avatar, appli, tampon)icone-* selon le fond
Impression noir & blanclogo-horizontal-noir
Document administratif sobrelogo-horizontal-monochrome

Format : SVG partout où c'est possible (net à toutes les tailles) ; PNG quand le SVG n'est pas accepté (réseaux sociaux, Word...).

3. Les 5 règles d'or

1. Ne jamais déformer, incliner ou recolorer le logo.  2. Respecter la zone de protection (hauteur du globe).  3. L'orange n'écrit jamais de texte courant sur fond blanc (accents et boutons seulement).  4. Titres en Poppins, texte en Inter/Open Sans — pas d'autres polices.  5. En dessous de 32 px, utiliser le favicon simplifié, pas l'icône complète.

4. Couleurs à copier-coller

Bleu #0A6EBD · Bleu foncé #084F86 · Orange #F59E0B · Gris texte #374151 · Blanc #FFFFFF

5. Modifier un support

Les sources des bannières, du post et des documents print sont des fichiers HTML dans les sous-dossiers src/ : ouvrez-les dans un éditeur, changez le texte, puis ré-exportez (capture ou impression PDF depuis le navigateur). Le logo source est le SVG — modifiable dans Inkscape, Figma ou tout éditeur vectoriel.
